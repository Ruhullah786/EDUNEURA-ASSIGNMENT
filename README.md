# 🌍 Wanderlust - Travel Listing Platform

## 📌 Project Overview

Wanderlust is a full-stack web application where users can explore, create, and manage travel listings. It allows users to share beautiful places, add details, and discover new destinations.

This project also integrates **AI-powered description generation**, making listing creation faster and smarter.

---

## 🚀 Features

* 🏡 Create, edit, and delete travel listings
* 📍 Location-based listings with Mapbox integration
* 🖼️ Image upload support
* 🤖 AI-generated descriptions (auto + manual)
* 🔐 User authentication (Login/Signup)
* ⭐ Reviews and ratings system
* 📱 Responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, Bootstrap, EJS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** Passport.js
* **Cloud Storage:** Cloudinary
* **Maps:** Mapbox
* **AI Integration:** Groq API / LLM

---

## 💡 AI Feature (Highlight)

Users can generate listing descriptions using AI by simply entering the title and location.
This improves user experience and saves time while creating listings.

## 📂 Project Structure

## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/wanderlust-project.git
```

2. Install dependencies:

```bash
npm install
```

3. Add `.env` file:

```
ATLASDB_URL=your_mongodb_url
SECRET=your_secret
MAPBOX_TOKEN=your_mapbox_token
GROQ_API_KEY=your_api_key
```

4. Run the server:

```bash
node app.js
```
